# phase_behaviors_near_criticality_A1LCD
All code and analyses to study near-critical vs off-critical phase behaviors of the prion-like low complexity domain A1-LCD 

How to run a Lassi simulation (energy and structure files ending with .prm should be in the same folder as the .key file):

./lassi -k param.key

*CSD* contains code to compute cluster size distributions, *cumulant* contains code to perform Binder cumulant analysis, *percolation_threshold* contains code to compute the percolation threshold from largest cluster size calculations,
